# phazzer
Fuzzer for Pharo


```smalltalk
RPackage organizer packages do: #phazz
```


# Fuzzing all tests in the image:

If you like to fuzz all tests installed in the image, run the following command:

```smalltalk
Phazzer hackThePlanet
```
